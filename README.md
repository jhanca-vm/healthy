# Healthy

## Getting Started

Install dependencies:

```bash
pnpm install
```

Create a `.env` file in the root of the project and add these variables:

```env
TURSO_DATABASE_URL
TURSO_AUTH_TOKEN
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.
